# Configuración de Red

## Objetivo

Configurar la conectividad del servidor y preparar el bridge para las futuras máquinas virtuales.

---

## Configuración

Bridge

vmbr0

IP

192.168.0.200

Gateway

192.168.0.1

---

## Bridge utilizado

vmbr0

Este bridge permitirá que todas las máquinas virtuales obtengan conectividad dentro de la red local.

---

## Validaciones

Se verificó:

- Acceso vía navegador.
- Comunicación con Internet.
- Resolución DNS.
- Actualización de paquetes.
